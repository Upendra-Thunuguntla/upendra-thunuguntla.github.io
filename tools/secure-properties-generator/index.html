FAQ

Frequently Asked Questions

Is this tool safe to use with production keys?
Yes. The encryption and decryption operations run entirely in your local browser using JavaScript. No data, keys, or configurations are ever sent to any remote server. It is completely safe.
Why does the result sometimes differ from the Mule tooling?
By default, standard Mule encryption uses a static initialization vector (IV). If you use the --use-random-iv flag in Mule, the output changes on every run. You can replicate this by checking the "Use random IV" box above!
What is RC4 / RCA?
RC4 is a stream cipher. Since it doesn't use blocks, it does not require an Operation Mode (CBC, ECB) or an Initialization Vector (IV). You'll notice those fields are disabled when RC4 is selected.
