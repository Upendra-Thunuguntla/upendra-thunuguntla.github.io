## Description
A fast, secure, and client-side Base64 converter. Encode and decode text or files directly in your browser without sending any data to external servers, protecting sensitive API credentials and payload logs.

## Key Features
- **Encode & Decode Modes:** Easily switch between encoding plain text/files to Base64 or decoding Base64 back.
- **Multiple Character Encodings:** Full support for UTF-8 (with standard Unicode emoji parsing), ASCII, Hexadecimal byte strings, UTF-16 Little Endian (LE), and UTF-16 Big Endian (BE).
- **URL-Safe Formatting:** Convert standard Base64 characters (`+`, `/`) to URL-safe options (`-`, `_`) and optionally strip padding `=` signs.
- **Line Wrapping Options:** Wrap outputs at 64 characters (MIME standard/PEM format) or 76 characters (email standards) to match configuration formats.
- **Detailed Parsing Analytics:** Instantly view input/output sizes, compression ratio percentages, line counts, and Base64 format validity flags.
- **100% Client-Side Engine:** Run all conversion algorithms locally in browser memory for zero server latency and absolute data security.
