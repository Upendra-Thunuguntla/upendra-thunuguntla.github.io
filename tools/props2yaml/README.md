# Properties to YAML Converter

## 🟢 Live Tool
**[Use Properties to YAML Converter →](https://upendra-thunuguntla.github.io/tools/props2yaml/)**

## Description
Free Properties to YAML converter for MuleSoft. Reconstruct nested YAML hierarchy from flat .properties files.

## Key Features
- **Unflattening Engine**: Transforms flat dotted keys (e.g. `db.port=3306`) into nested indentation-based YAML nodes.
- **MuleSoft Optimized**: Designed for MuleSoft application properties structuring.
- **Client-Side Security**: Safe, offline-capable conversion that respects your configuration security.
