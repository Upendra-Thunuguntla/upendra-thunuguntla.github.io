# YAML to Properties Converter

## 🟢 Live Tool
**[Use YAML to Properties Converter →](https://upendra-thunuguntla.github.io/tools/yaml2props/)**

## Description
Free YAML to Properties converter for MuleSoft. Flatten YAML configs into flat properties, extract keys with ${key} or Mule::p() wrapping, and support secure properties prefix.

## Key Features
- **Flattening Engine**: Convert structured, nested YAML files into dot-notated property chains.
- **Multiple Formats**: Easily generate keys only or wrap them in `${key}` or `Mule::p("key")` functions for MuleSoft and DataWeave.
- **Secure Key Detection**: Automatically appends `secure::` prefixes to keys mapped to encrypted values (marked by `![value]`).
- **Privacy Guaranteed**: Runs completely offline in the browser to protect production database credentials and application settings.
