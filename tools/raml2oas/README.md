# RAML to OAS (OpenAPI) Converter

## 🟢 Live Tool
**[Use RAML to OAS (OpenAPI) Converter →](https://upendra-thunuguntla.github.io/tools/raml2oas/)**

## Description
Convert RAML 1.0 API specifications to OpenAPI 3.0 YAML or JSON instantly. 100% browser-based.

## Key Features
- **RAML 1.0 to OAS 3.0**: Seamlessly convert your MuleSoft API specifications to standard OpenAPI format.
- **YAML & JSON Output**: Supports download of converted specifications in both YAML and JSON styles.
- **Client-Side Rendering**: Parses specifications entirely in the browser to ensure your API designs stay private.
