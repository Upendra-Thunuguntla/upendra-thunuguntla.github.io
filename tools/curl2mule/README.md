# Postman cURL to Mule Converter

## 🟢 Live Tool
**[Use Postman cURL to Mule Converter →](https://upendra-thunuguntla.github.io/tools/curl2mule/)**

## Description
Instantly convert cURL commands (from Postman, DevTools, or terminal) into MuleSoft HTTP Request Configuration and Processor XML. 100% browser-based and secure.

## Key Features
- **cURL Parser**: Automatically extracts method, URL, headers, query parameters, and request body from standard cURL commands.
- **XML Generator**: Generates both the `http:request-config` and `http:request` Mule XML blocks.
- **DataWeave Mapping**: Translates request bodies and headers to appropriate Mule expressions.
- **100% Client-Side**: Safe, secure, and requires no network requests, maintaining confidentiality of your API payloads.
