## Description
Free online JSON to RAML DataType converter for MuleSoft API Designer. Supports optional keys, CamelCase conversion, and RAML 1.0 format.

## Key Features
- **Instant DataType Generation**: Convert raw JSON payloads into fully formatted RAML 1.0 DataTypes.
- **CamelCase Conversion**: Standardize keys to camelCase automatically.
- **Optional Keys**: Identify and configure optional property flags.
- **100% Client-Side**: No data leaves your browser, ensuring absolute security and privacy.
