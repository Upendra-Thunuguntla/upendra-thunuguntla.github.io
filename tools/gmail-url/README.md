# Gmail Compose URL Generator

## 🟢 Live Tool
**[Use Gmail Compose URL Generator →](https://upendra-thunuguntla.github.io/tools/gmail-url/)**

## Description
Generate pre-filled Gmail compose URLs with To, CC, BCC, Subject, and Body. Bookmark and reuse for recurring emails.

## Key Features
- **Field Encoding**: Correctly URI-encodes all fields (To, CC, BCC, Subject, Body) to generate valid web-safe links.
- **Save & Reuse**: Create bookmarks of your generated links to launch Gmail compose setups instantly.
- **Privacy Focused**: Operates entirely client-side without storing or transmitting any email details.
